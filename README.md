resource-monitor

Projeto RA3: Sistema de Monitoramento, Profiling e Análise de Recursos no Linux.

Esta pasta contém a implementação inicial exigida pela disciplina: coletor de recursos,
analisador de namespaces e gerenciador de cgroups. Os arquivos principais estão em
`src/`, as interfaces em `include/`, documentação em `docs/` e scripts auxiliares em `scripts/`.

Siga o `Makefile` para compilar e rodar testes básicos.

Estrutura exigida pelo enunciado já mapeada aqui.

---

Para detalhes e metodologia dos experimentos, veja `docs/ARCHITECTURE.md`.
